# sber_task
Скрипт на python. 

Важное замечание: 
Задании сказано "найти среднюю доходность....агрегированную по...приложенному списку клиентов". 
В ходе выполнения задания было обнаружено, что только 125 клиентов из 200 уникальных клиентов в id_clients.xlsx есть в базе данных. 
После объединения данных методом inner join базы и фильтрации клиентов по сегменту бизнеса осталось 110 клиентов из 200.
Отсутствующие клиенты не учитываются в среднем и не считаются клиентами с нулевой доходностью. Среднее рассчитывается только по найденным данным.
