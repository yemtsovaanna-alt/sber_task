### Важное замечание

Важное замечание: в задании сказано "найти среднюю доходность....агрегированную по...приложенному списку клиентов".<br>
В ходе выполнения задания было обнаружено, что только 125 клиентов из 200 уникальных клиентов из id_clients.xlsx есть в базе данных.<br>
После объединения данных методом inner join базы и фильтрации клиентов по сегменту бизнеса осталось 110 клиентов из 200.<br>
Отсутствующие клиенты не учитываются в среднем и не считаются клиентами с нулевой доходностью.<br>
Среднее рассчитывается только по найденным данным.
